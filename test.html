там внизу есть ещё...

Список моих проектов

В списке только самые актуальные проекты или те, над которыми я долго и упорно работал. Остальное на моем Github.


  • devreader

    Простой, удобный, а также мобильный справочник по программированию для начинающих

  • devreader/CoreX

    Минималистичная, легкая и простая библиотека для верстки небольших веб-сайтов (и не только), написанная на SASS с минимальным кодом JavaScript

  • rx1310/a2iga

    Маленькая утилита, которая поможет заменить Google Assistant, вызываемый зажатием кнопки Home / жестом нижних краёв экрана (Android Q), на любое приложение, нужное вам

  • rx1310/RebootManager

    Простой, лёгкий и удобный менеджер перезагрузки вашего Android-устройства в режим recovery и bootloader

  • rx1310/ChangeDPI

    Маленькая утилита для смены DPI на устройстве (required superuser permission)

  • Давно не поддерживаются
  • rx1310/VanillaCoreX

    Минималистичный, простой, легкий и красивый фреймворк для верстки простых веб-сайтов, написанный с использованием чистого JavaScript и CSS

  • rx1310/novax

    Одна из самых масштабных модификаций Nova Launcher для Android
