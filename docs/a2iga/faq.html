#1: установка

думаю с установкой apk a2iga вы сами смогли разобраться, а вот с настройкой приложения-помощника в системе вам поможет само приложение.

при первом запуске a2iga выведет небольшой блок с уведомлением о том, что a2iga не является приложением-помощником в системе по умолчанию. от вас требуется нажать на эту карточку (синего цвета) > перейти в раздел «помощник» > поставить галочку рядом с a2iga > все.

#2: appslist

appslist - это специальный список установленных на устройстве приложений.

appslist нужен для более удобного выбора приложения, которое будет запускаться при вызове ассистента.

если количество приложений на устройстве мешает искать нужное приложение в списке, то для таких случаев в appslist предусмотрен поиск по названию приложения.

чтобы выбрать нужное приложение в качестве ассистента нужно:

  1. найти нужное приложение в списке
  2. нажать на пункт с названием этого приложения
  3. подтвердить установку

все очень просто. настроив a2iga один раз, вы намного ускорите доступ к вашему любимому приложению. больше не потребуется каждый раз сворачивать приложение и открывать меню приложений чтобы запустить ваш любимый мессенджер / игру / плеер / ... / браузер!

#3: вызов ассистента

по сути, вызов ассистента на разных устройствах может быть разным. где-то это удержание кнопки «home», где-то недлительное удержание кнопки «power», на pixel'ях 2/3 это сжатие по бокам.

в основном ассистент запускается длительным удержанием кнопки «home» (кружок по центру на aosp).
